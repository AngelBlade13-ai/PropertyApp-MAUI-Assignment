## Demo Evidence

Record a 1-3 minute video or capture screenshots showing:

- [ ] App running on an emulator/device.
- [ ] Landing page property list.
- [ ] Property card selection.
- [ ] Details page navigation.
- [ ] Details page image, thumbnails, price, and address.
- [ ] Close/back navigation.

## Suggested Video Flow

1. Start on the landing page.
2. Point out the New York listing screen and category buttons.
3. Tap one property card.
4. Show the details page with the large property image.
5. Show the extra thumbnail images and property details.
6. Tap the close button to return to the landing page.

## Notes

- This app does not use a database, so no migration setup is required.
- Remote images require internet access.
- Warnings are currently present because the tutorial uses some APIs that are deprecated in newer MAUI versions, but the app should build and run.
